# 🚀 MicroMaker

A full-stack  web app for hunting micro-tasks around the world.

👉 **[Live Demo](https://micro-maker.vercel.app/)**

---

## ✨ Features
- ⚡ Admins can manage users and tasks.
- ⚡ Workers may serach for tasks and earn money which they can withdraw through stripe payout.
- ⚡ Buyers may add tasks and buy coins through stripe to pay for their tasks.


---

## 🧰 Tech Stack

**Frontend:**
- NextJs
- Tailwind CSS
- Swiper

**Backend:**
- Node.js  
- PostgreSql & Supabase

**Authentication:**
- NextAuth  

**Deployment:**
- Vercel
